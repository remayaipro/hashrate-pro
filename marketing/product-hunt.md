# HashRate Pro - Product Hunt Launch

## Name
HashRate Pro - Crypto Mining Profitability Calculator

## Tagline
Real-time mining profitability calculator with live prices. GPU + ASIC support.

## Description

### Short Description
Free crypto mining calculator with live CoinGecko prices. Compare GPU and ASIC miners, calculate daily/monthly/yearly profit after electricity costs. No signup required.

### Full Description
HashRate Pro helps crypto miners make smarter decisions with real-time profitability calculations.

**What it does:**
- Calculates mining profit for 8 mineable coins (BTC, LTC, DOGE, KAS, ETC, XMR, RVN, ERG)
- Supports both GPU and ASIC miners
- Live prices from CoinGecko (updates every 60 seconds)
- Hardware comparison table with profit estimates
- Customizable electricity costs
- Pool fee calculations
- Break-even analysis

**Why I built it:**
Most mining calculators are outdated, slow, or require signup. I wanted something fast, accurate, and free.

**Who it's for:**
- Hobby miners deciding if mining is worth it
- Miners comparing hardware purchases
- Anyone curious about crypto mining profitability

**Tech stack:**
- Pure HTML/CSS/JS (no framework, loads instantly)
- CoinGecko free API for live prices
- Cloudflare Pages for global CDN
- Dark glassmorphism theme

**Completely free. No signup. No tracking. No BS.**
